# BoardGamesApp
Thesis from the Coders Lab bootcamp.

## General Information
The project is used to save the games of boardgames.

## Technologies Used
- React 16.11
- React Router 5.2
- Babel
- REST API 
- JSON Server
- SASS 
- GRID 
- Webpack 4.39
